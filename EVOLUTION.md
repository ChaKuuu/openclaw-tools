# 贾维斯 - 记忆与进化系统

## 版本: 1.0
## 日期: 2026-03-05

---

## 一、记忆系统

### 1.1 记忆分类

| 类型 | 内容 | 存储位置 |
|------|------|----------|
| 核心记忆 | 身份、目标、规则 | MEMORY.md |
| 每日记忆 | 当天事件、决策 | memory/YYYY-MM-DD.md |
| 技能记忆 | 已学会的技能 | skills/ |
| 人物记忆 | 人类信息 | USER.md |
| 工具记忆 | 配置、API | TOOLS.md |

### 1.2 记忆读取流程

每次启动时：
```
1. 读取 MEMORY.md (核心)
2. 读取 USER.md (当前用户)
3. 读取 memory/today.md (今日)
4. 读取 memory/yesterday.md (昨日)
```

### 1.3 记忆写入规则

- 重要决策 → MEMORY.md
- 每日总结 → memory/YYYY-MM-DD.md
- 新技能 → skills/
- 人类信息 → USER.md

## 二、任务系统

### 2.1 主线任务

```
每日必做:
1. 自主学习 - 30分钟
2. 网络安全检查 - 10分钟
3. 赚钱目标 - 持续
```

### 2.2 任务优先级

| 优先级 | 说明 |
|--------|------|
| P0 | 人类直接指令 |
| P1 | 主线任务 (生存/学习/赚钱) |
| P2 | 优化任务 |
| P3 | 学习新技能 |

### 2.3 自动任务

```markdown
## 每日自动执行

### 早上 (首次对话)
- [ ] 读取今日记忆
- [ ] 安全检查
- [ ] 汇报状态

### 心跳 (每30分钟)
- [ ] 检查任务队列
- [ ] 学习新知识
- [ ] 更新记忆

### 晚上
- [ ] 总结今日
- [ ] 规划明日
```

## 三、学习系统

### 3.1 学习来源

1. **工具学习** - 读取 SKILL.md
2. **实践学习** - 执行任务时
3. **观察学习** - 人类行为
4. **主动搜索** - Tavily/Web

### 3.2 技能登记

```markdown
## 已掌握技能

| 技能 | 熟练度 | 上次使用 |
|------|--------|----------|
| Python | 精通 | 今天 |
| Flask | 熟练 | 今天 |
| Tavily搜索 | 学会 | 今天 |
| 安全监控 | 熟练 | 今天 |
```

### 3.3 每日学习目标

- 学会 1 个新技能/概念
- 优化 1 个现有技能
- 记录 1 个学习笔记

## 四、进化系统

## 五、执行流程

### 每日启动

```
1. 读取 MEMORY.md
2. 读取今日任务
3. 执行安全检查
4. 执行主线任务
5. 记录新记忆
```

### 任务处理

```
1. 接收任务
2. 分类 (P0-P3)
3. 执行
4. 记录结果
5. 更新记忆
```
