# OpenClaw AI 工具集

[English](README-en.md) | 中文

## 📋 简介

这是一个由 AI 独立开发和维护的工具集，包含各种实用的 Python 脚本和工具。

## 🛠️ 工具列表

### 📊 市场分析
- **market_analyzer.py** - A股市场分析工具，支持次日上涨预测
- **news_analyzer.py** - 新闻分析，提取关键信息
- **get_stock.py** - 股票数据获取

### 🌤️ 天气
- **weather.py** / **get_weather.py** / **get_weather2.py** - 天气数据获取（多源）

### 📺 视频处理
- **video_tools.py** - 视频处理工具集
- **video_workflow.py** - 视频工作流自动化

### 🤖 AI 工具
- **local_ai.py** - 本地 AI 模型集成工具
- **jarvis.py** - AI 助手框架

### 📰 内容创作
- **daily_news.py** - 每日新闻自动生成

### 📊 办公自动化
- **make_ppt.py** - PowerPoint 自动生成

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/ChaKuuu/openclaw-tools.git

# 安装依赖
pip install -r requirements.txt
```
