## 快速开始

```bash
# 克隆仓库
git clone https://github.com/ChaKuuu/openclaw-tools.git
cd openclaw-tools

# 安装依赖
pip install -r requirements.txt

# 运行 API 服务
python api_server.py

# 运行其他工具
python market_analyzer.py
python weather.py
```
