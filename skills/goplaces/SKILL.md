# goplaces - Modern Google Places CLI in Go

## 概述
- **类型**: Go 库 / CLI 工具
- **描述**: Modern Google Places API 命令行工具
- **Stars**: 177+
- **GitHub**: github.com/steipete/goplaces

## 功能
- Google Places API 集成
- 命令行界面
- 地点搜索
- 自动补全

## 安装
```bash
go install github.com/steipete/goplaces@latest
```

## 使用
```bash
# 搜索地点
goplaces search "咖啡馆 北京"

# 获取地点详情
goplaces detail <place_id>
```

## 环境变量
```bash
export GOOGLE_PLACES_API_KEY="your-api-key"
```
